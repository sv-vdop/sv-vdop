**Номер:** BUG-3

**Название:** Избранное: счётчик иконки не отображает актуальное количество товаров при переходе между страницами

**Статус:** Новый

**Приоритет:** Средний

**Шаги:**

1. Открыть страницу https://lazurit.com/product/6218664/divan-uglovoy-levyy-pekin-lambre05/
2. Добавить товар в избранное.
3. Убедиться, что счётчик у иконки избранного показывает «1».
4. Перейти на любую другую страницу сайта.

**Ожидаемый результат:** Счётчик у иконки избранного отображается на всех страницах и показывает актуальное количество товаров.

**Фактический результат:** Счётчик у иконки избранного не отображается. При этом запрос GET /api/v2/favorites/?cityId=2 возвращает статус 200 OK и тело ответа с "count": 1 — сервер данные отдаёт корректно. Баг на стороне фронтенда.
