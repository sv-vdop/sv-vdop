**Название:** Промокод: кнопка "Удалить" промокод удаляется за 5–30 секунд вместо 1–2  
**Статус:** Новый  
**Приоритет:** Низкий  

**Шаги:**

1. Добавить в корзину товар по ссылке https://lazurit.com/product/6207166/tumba-ilona-313949/.
2. Открыть корзину.
3. Ввести валидный или невалидный промокод в поле ввода промокода.
4. Нажать кнопку "Применить".
5. Нажать кнопку "Удалить".

**Ожидаемый результат:** Удаление в течение 1–2 секунд.  
**Фактический результат:** Ожидание удаления 5–30 секунд.
