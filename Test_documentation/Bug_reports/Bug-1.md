**Номер:** BUG-1  

**Название:** Промокод: кнопка "Удалить" долго удаляет промокод

**Статус:** Новый  

**Приоритет:** Низкий  

**Шаги:**

1. Добавить в корзину товар по ссылке https://lazurit.com/product/6207166/tumba-ilona-313949/
2. Открыть корзину
3. Ввести валидный или невалидный промокод в поле ввода промокода
4. Нажать кнопку "Применить"
5. Нажать кнопку "Удалить"

**Ожидаемый результат:** Удаление без ожидания

**Фактический результат:** Удаление спустя 5–30 секунд
