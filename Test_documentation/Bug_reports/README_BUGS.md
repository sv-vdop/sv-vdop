# Баг-репорты — lazurit.com

Папка содержит баг-репорты, найденные в ходе ручного тестирования сайта [lazurit.com](https://lazurit.com).  
Тестировались модули: форма регистрации, корзина, избранное, каталог товаров.

**Тестировщик:** Alexander V.

| [BUG-4](Bug-4.md) | Каталог "Кровати": при выборе всех значений фильтра "Подъёмный механизм" отображается меньше товаров, чем есть в категории | Низкий
