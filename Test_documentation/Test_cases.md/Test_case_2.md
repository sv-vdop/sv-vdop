**Номер:** 2

**Название:** Быстродействие кнопки удаления промокода в корзине.

**Шаги:**
1. Добавить в корзину любой товар на сайте в любом количестве. 
2. Открыть корзину. 
3. Ввести любой текст в поле ввода промокода. 
4. Нажать кнопку "Применить".
5. Нажать кнопку "Удалить".

**Ожидаемый результат:** Промокод удалится в течение 1-2 секунд.
