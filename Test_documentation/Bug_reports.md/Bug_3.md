**Номер:** 3

**Название:** Удаление нескольких товаров из корзины с переходом на другую страницу.

**Статус:** Новый

**Приоритет:** Низкий

**Шаги:**
1. Добавить в корзину любые 4 товара. 
2. Открыть корзину. 
3. Нажать "Удалить" у 3 товаров. 
4. Не ожидая несколько секунд, пока товар удаляется, перейти на любую другую страницу сайта. 
5. Перейти обратно в корзину.

**Фактический результат:** Удалился только 1 товар.

**Ожидаемый результат:** Удалятся все товары, у которых была нажата кнопка "Удалить".
